This module will help you compute the sales tax for all the sales order.

User can select a defined date range, and automatically the computation will be done.

In order to compute the sales taxes, use the menu Reporting > Tax Computation in the accounting application. From this menu, you will get all the sales taxes amount of all the Invoice.

It also tell you about the Number of sales that had been done during a particular time period. This module is specially for Indian Customers where tax is applicable on goods sale/purchase and on Invoicing, CIS offering this module so that It will be easy to compute sales taxes on Invoicing.
